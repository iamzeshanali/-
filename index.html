Clean Pakistan (Smart Garbage Collection)

Final Year Project.
Part of Clean Green Pakistan Initiative led by PM Imran Khan

The IOT Based Project developed in AngularJS( Angular Material) and Firebase Firestore. A solution to severe issues of pollution caused by ill-efforts of cleansing of streets and cities by Munciple Committites in Pakistan.

This garbage collection system is developed for dustbin clearance by giving an alert notification to the municipal web server for instant cleaning of a dustbin. This alert notification is generated based on the level of garbage in the bin. An ultrasonic sensor which is interfaced with Arduino esp8266 is used to check the level of garbage in bin. The notification will be sent to the municipal web server through which it will be delivered to user by an android application along with garbage bin number and its location once if garbage is 80% filled. Once the alert is received, Municipal Corporation takes initiative to clean the garbage bin. After cleaning the garbage bin, the municipal web server gets updated about the garbage bin been cleaned. This system provides the information that whether the dustbins are cleaned up in proper time or not and how municipal committee cope with these issues.

Scope includes building, testing and deployment of the following apps:

- Super Admin Web
- Account Holder Web
- Android Apis

Life cycle of the entire platform should work in a way that users perform signup and their accounts are verified by Super Admin. First Admin assign role of Supervisor to the certain users and those users assign role of workes to other users. Workers perform their duties and their progress report with different time spans is shown to the supervisors and administrator.
